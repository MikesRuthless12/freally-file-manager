Freally File Manager 0.20.0 — Latest release

The complete byte-exact copy engine, desktop app, and CLI — verify-after-copy, pause/resume, two-way sync, secure delete, and cloud backends, all in one release.

What's included

  • Byte-exact copy engine with verify-after-copy (xxHash3, BLAKE3, SHA-256), pause/resume/cancel, and collision policies (Keep both, Skip, Overwrite, Overwrite if newer, Ask).
  • Secure multi-pass file shredding: Zero, Random (CSPRNG), DoD 3-pass, DoD 7-pass, Schneier 7-pass, VSITR 7-pass, Gutmann 35-pass, NIST 800-88 Clear — with SSD advisory.
  • Platform fast-paths: reflink/clonefile, CopyFileExW, copy_file_range(2), sendfile — auto-selected per filesystem.
  • Shell integration on every platform: Windows IExplorerCommand, macOS Finder Sync + Services, Nautilus, Dolphin ServiceMenu, Thunar Custom Actions.
  • Global paste hotkey (Ctrl+Shift+V / Cmd+Shift+V), clipboard watcher, history & totals, profiles, and 18-language UI.
  • 12 cloud backends via OpenDAL + SFTP: S3, GCS, Azure Blob, Backblaze B2, Cloudflare R2, DigitalOcean Spaces, Wasabi, MinIO, WebDAV, FTP, and more.
  • 2 429 MiB/s throughput on a modern NVMe — +86 % faster than Robocopy on the same hardware.

Roadmap

  1. Shipped
    Per-OS installers & signed auto-updater

    MSI / NSIS (Windows), DMG (macOS — Intel + Apple Silicon), deb / rpm / AppImage (Linux), plus a free ed25519-signed auto-updater. Shipped in 0.20.0.

    Shipped · 0.20.0
  2. Shipped
    Server mode & observability

    Run Freally File Manager headless as a WebDAV / HTTP / S3 / SFTP file server over any folder — read-only mode, bearer or HTTP-Basic auth, AWS SigV4 on the S3 API, a Prometheus /metrics endpoint, OpenTelemetry trace export, and Slack / Discord / ntfy.sh / Pushover webhooks. Launch it with the freally serve command, or set it up in the new Settings → Server panel.

    Shipped · 0.20.0
  3. Shipped
    Mount-as-filesystem, enterprise audit log, encryption & compression

    Mount any cloud backend or snapshot as a local drive (FUSE / WinFsp), export a tamper-evident WORM audit log in Syslog / CEF / LEEF / CSV / JSON formats, and encrypt (age) or compress (zstd) on the fly.

    Shipped · 0.20.0
  4. Shipped
    Sandboxed post-copy plugins

    Run third-party WebAssembly plugins after a copy, move, or job — fuel- and memory-capped, wall-clock-bounded, and gated by explicit per-plugin capabilities. Install from a file or hash-pinned URL in Settings → Plugins, with sample plugins for EXIF-based organizing and Discord / ntfy notifications.

    Shipped · 0.20.0
  5. Shipped
    Named queues & drag-merge

    Group transfers into named queues per drive, drag one queue's tab onto another to merge their progress into a single operation, press F2 to auto-enqueue the next job, and pin tray destinations for one-drop copies.

    Shipped · 0.20.0
  6. Shipped
    Live copy diagnostics

    When a transfer slows down, a header badge names the bottleneck — source disk, destination disk, network, antivirus, CPU, or thermal throttling — with a live throughput sparkline, backed by a real per-OS disk-busy sampler.

    Shipped · 0.20.0
  7. Next
    Signed builds

    Azure Trusted Signing on Windows and Apple notarization on macOS — no more SmartScreen or Gatekeeper prompts.

    Planned
  8. Beyond

    Interactive three-way merge for sync conflicts, plus expanded WASI host APIs so plugins can read and write files directly under their granted capabilities.
